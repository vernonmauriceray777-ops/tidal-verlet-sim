# tidal-verlet-sim
Stable 3-body tidal simulation in Python. Velocity Verlet + Plummer softening + spin-orbit torque. Energy conserved to 1e-14.
 Black Hole Simultor Lense-Thirring Precession Simulation N-body simulation of test particles orbiting a rotating black hole. Includes Newtonian vs GR frame-dragging force comparison and residual precession plots.
