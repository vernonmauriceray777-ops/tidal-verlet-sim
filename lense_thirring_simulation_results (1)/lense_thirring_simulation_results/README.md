
# Lense-Thirring Precession Simulation

This notebook contains an N-body simulation demonstrating Lense-Thirring precession around a rotating black hole.

## Contents:
- `lense_thirring_sim.ipynb`: The main Colab notebook with the simulation code.
- `README.md`: This file.
- `plots/gr_residual.png`: Plot showing the residual orbital angle with General Relativistic physics (frame-dragging).
- `plots/newton_residual.png`: Plot showing the residual orbital angle with Newtonian physics (no frame-dragging).

To run the notebook, open it in Google Colab.
